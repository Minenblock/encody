# Welcome to encody!

A simple CLI-Tool that encrypts a File to a Hash (AES-GCM) with a given Password.


# How to use
- download java (jdk 17/21 should work)
- git clone https://github.com/Minenblock/encody or download from Releases
- run the encody class or run the jar file with "java -jar [path/to/file].jar"

# Features
- select the file you want to encrypt with an password
- if you run the jar again and select the encrypted file you can decrypt it with the same password
